LEVEL REGISTER

komponen yang lebih kompak dibandingkan dengan sebuah gerbang logik dan flip-flop.

definisi register

apa itu register

Register merupakan memori pada bagian komputer (mikroprosessor) yang berkapasitas kecil namun memiliki kecepatan baca yang sangat tinggi.

Register pada mikroprosessor dapat diibaratkan sebagai kaki dan tangan dari mikroprosessor, karena dalam setiap pekerjaan, mikroprosessor selalu mengandalkan dan melibatkan register sebagai perantara dan sebagai komponen yang paling banyak melakukan pekerjaan mikroprosessor

Secara umum, register berfungsi sebagai tempat penyimpanan sementara untuk menyimpan hasil dari operasi aritmatika ataupun operasi logika yang dilakukan oleh mikroprosessor

8 Register Berdasarkan Data yang Dikelompokkan

Register Data

Register Alamat

Menyimpan alamat yang digunakan untuk mengakses memori

Register Tujuan Umum

Menyimapan data maupun alamat

Register Floating Point

Menyimpan bilangan floating point

Register Konstanta

Menyimpan nilai yang hanya dapat dibaca

Register Vektor

Menyimpan data untuk melakukan pemrosesan vektor

Register Tujuan Khusus

Menyimapan kondisi program termasuk didalamnya pencacah program stack point dan register status

Register Berhubungan dengan akses memori

Berhubungan dengan pengaksesan memori seperti register penahan, Register data dan register alamat

Jenis Rangkaian Komponen Register

Rangkaian Kombinasional

Setiap outputnya hanya merupakan fungsi input pada suatu saat tertentu saja.
Komponen: Multiplexer, Demultiplexer, Decoder, Encoder, dan Arithmetic Elements

Rangkaian Sekuensial

Setiap outputnya tidak hanya tergantung pada input waktu itu saja, tetapi juga pada keadaan input sebelumnya.
Komponen: Paralel Register, Shift Register, dan Counter.

Register geser

apa itu register geser?

register yang dapat melakukan penggeseran ke kiri dan ke kanan
Untuk melakukan penggeseran, harus mempunyai m buah flip-flop master-slave yang masing masingnya dihubungkan ke sebelah kiri kanannya

Berguna untuk menyimpan data serial, mengonversi data serial ke paralel atau sebaliknya, dan melakukan operasi aritmatika

pencacah (counter)

apa itu pencacah?

Sebuah mesin sekuensial yang dirancang untuk melakukan pencacahan

Dapat diperoleh dengan sedikit melakukan modifikasi dari register geser
